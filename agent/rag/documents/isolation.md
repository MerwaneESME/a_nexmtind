## MATÉRIAUX COURANTS

### Caractéristiques thermiques

**Lambda (λ) - Conductivité thermique** (W/m.K)
- Plus λ est faible, plus c'est isolant
- **Excellents** : λ < 0,030 (PUR, PIR)
- **Très bons** : λ 0,030-0,040 (PSE, XPS, laines)
- **Bons** : λ 0,040-0,050 (naturels)

**Résistance thermique R** (m².K/W)
- R = épaisseur / λ
- Plus R est élevé, meilleure isolation
- **RE2020 toiture** : R ≥ 8 m².K/W
- **RE2020 murs** : R ≥ 4 m².K/W
- **RE2020 planchers** : R ≥ 4 m².K/W

### Critères de choix

**Performance thermique**
- Combles : PUR, PIR, laine de verre épaisse
- Murs intérieurs : PSE, laine de roche
- ITE : PSE graphité, laine de roche, fibre de bois

**Résistance au feu**
- **Incombustibles** : Laines minérales (A1, A2)
- **Combustibles traités** : PSE classé E
- **Zones à risque** : Privilégier laine de roche

**Acoustique**
- **Excellent** : Laine de roche, fibre de bois
- **Bon** : Laine de verre, ouate de cellulose
- **Faible** : PSE, XPS, PUR

**Humidité**
- **Hydrophiles** : Laines minérales (pare-vapeur obligatoire)
- **Hydrophobes** : XPS, PUR (sous-sol, soubassement)
- **Perspireants** : Naturels (fibre bois, chanvre)

**Écologie**
- **Biosourcés** : Bois, cellulose, chanvre, laine de mouton
- **Recyclables** : Laine de verre, ouate de cellulose
- **Énergie grise faible** : Ouate, chanvre, laine de mouton

## ERREURS FRÉQUENTES

### Conception
❌ **Épaisseur insuffisante**
- R ne respectant pas RE2020
- → Performances insuffisantes, inéligibilité aides

❌ **Isolant inadapté à l'usage**
- PSE en zone humide
- Laine sans pare-vapeur
- → Dégradation, perte performance

❌ **Oubli pare-vapeur/frein-vapeur**
- Combles avec laine minérale
- → Condensation, moisissures

❌ **Absence de ventilation**
- Isolation sans renouvellement d'air
- → Humidité, qualité d'air dégradée

### Mise en œuvre

❌ **Ponts thermiques non traités**
- Discontinuité isolant
- Fixations métalliques traversantes
- → Déperditions 20-30%

❌ **Compression de l'isolant**
- Laine tassée
- → Perte performance 50%

❌ **Jointoiement insuffisant**
- Lames d'air entre panneaux
- → Convection, perte 15-25%

❌ **Pare-vapeur côté froid**
- Inversion sens de pose
- → Condensation dans isolant

❌ **Soufflage irrégulier**
- Épaisseur variable combles
- → Zones froides

### Finitions

❌ **Suppression ventilation existante**
- Obturation grilles, VMC
- → Problèmes condensation

❌ **Enduit fissuré (ITE)**
- Sous-couche insuffisante
- Treillis mal positionné
- → Infiltrations

## CONSEILS PRATIQUES

### Avant travaux

✓ **Diagnostic thermique**
- Audit énergétique recommandé
- Identifier priorités (toiture 30% pertes)
- Simuler aides financières

✓ **Ventilation**
- Vérifier système adapté (VMC)
- Prévoir installation si absente
- Jamais isoler sans ventiler

✓ **Traiter humidité**
- Fuites toiture
- Remontées capillaires
- Condensation existante

✓ **Choix isolant**
- Performance vs budget
- Contraintes épaisseur
- Objectif certification (BBC, Passif)

### Pendant travaux

✓ **Étanchéité à l'air**
- Jointoyer tous panneaux
- Adhésifs spécifiques pare-vapeur
- Traiter passages câbles/gaines

✓ **Continuité isolation**
- Traiter jonctions (mur/plancher, mur/toiture)
- Retours tableaux fenêtres 10cm minimum
- Isoler coffres volets roulants

✓ **Protection**
- Laines minérales : EPI (gants, masque, lunettes)
- Stockage au sec
- Bâchage si interruption chantier

✓ **Contrôle qualité**
- Vérifier épaisseur (pige graduée)
- Test à la thermographie après pose
- Photos avant fermeture (litiges)

### Points spécifiques

**Combles perdus**
- Épaisseur 300-400mm (R≥8)
- Éviter circulation sur isolant
- Repères hauteur trappe visite
- Protection spots encastrés (écart 10cm)

**ITE**
- Traiter tous ponts thermiques
- Profils départ, angles, ouvertures
- 2 couches croisées si possible
- Finition adaptée climat

**Planchers bas**
- Isoler sur toute surface
- Attention hauteur sous plafond
- Isolation + pare-vapeur si terre-plein

**Rénovation BBC**
- R ≥ 8 toiture, R ≥ 4 murs, R ≥ 4 planchers
- Étanchéité à l'air Q4Pa < 0,6 m³/h.m²
- Test infiltrométrie obligatoire

## MINI-RÉSUMÉS DTU

### DTU 45.10 (Isolation combles)

**Combles perdus**
- **Épaisseur mini RE2020** : R ≥ 8 m².K/W (~400mm laine verre)
- **Soufflage** : Épaisseur +10% compte tenu tassement
- **Déroulage** : 2 couches croisées recommandées
- **Passage conduits** : Écart 10cm mini spots/isolant

**Combles aménagés**
- **Rampants** : R ≥ 6-8 m².K/W
- **Pare-vapeur obligatoire** : Sd ≥ 18m (laines minérales)
- **Lame air** : 2cm mini sous couverture (ventilation)
- **Fixation** : Agrafage tous 10-15cm, recouvrement 10cm

**Ventilation**
- Entrées air en façade maintenues
- VMC : débit adapté au volume isolé
- Chatières toiture conservées

### DTU 20.1 (Murs en maçonnerie)

**Isolation intérieure**
- **Doublage collé** : Plots mortier-colle, espacement 30-40cm
- **Contre-cloison** : Ossature, vide air 2cm mini
- **Pare-vapeur** : Si laine minérale, Sd ≥ 18m
- **Épaisseur mini** : R ≥ 3,7 m².K/W (RE2020 → R ≥ 4)

**Points singuliers**
- Retours tableaux : 10cm mini
- Seuils, linteaux : Traitement continuité
- Coffres volets : Isolation spécifique

### DTU 31.2 (Maisons ossature bois)

**Isolation MOB**
- **Remplissage** : Isolant semi-rigide entre montants
- **Complément extérieur** : +40-60mm (ponts thermiques)
- **Pare-vapeur intérieur** : Continu, Sd ≥ 18m
- **Pare-pluie extérieur** : Sd ≤ 0,5m (perspirant)

**Étanchéité air**
- Jonctions pare-vapeur : Adhésifs/mastics
- Passages réseaux : Manchons étanches
- Objectif Q4 : < 0,6 m³/h.m²

### CPT 3035 (ITE sous enduit)

**Supports admis**
- Maçonnerie, béton, enduit ancien sain
- Planéité 10mm/2m
- Résistance arrachement > 0,3 MPa

**Mise en œuvre**
- **Fixation** : Collage + chevilles (5-8/m²)
- **Épaisseur** : 120-200mm courant
- **Sous-couche** : Armée treillis fibre verre
- **Finition** : Enduit organique ou minéral

**Joints**
- Panneaux décalés (joints croisés)
- Baguettes angles, départ, arrêt
- Profilés fenêtres avec goutte d'eau

### NF DTU 27.1 (Bardage rapporté)

**ITE + bardage**
- **Structure** : Ossature bois ou métal
- **Lame air ventilée** : 20mm mini
- **Pare-pluie** : Obligatoire sur isolant
- **Grilles anti-rongeurs** : Bas et haut

## POINTS DE VIGILANCE 2025-2026

### Réglementation

**RE2020 renforcée**
- Seuils R minimaux obligatoires
- Calcul impact carbone matériaux
- Biosourcés favorisés (bonus)

**Aides conditionnées**
- Entreprise RGE obligatoire
- Performance minimale (R ≥ 7 combles, R ≥ 3,7 murs)
- Audit énergétique si rénovation globale

### Évolutions techniques

**Isolants nouvelle génération**
- Aérogels (λ 0,015) en développement
- PIR ultra-performants (λ < 0,022)
- Biosourcés certifiés (lin, paille, algues)

**Membranes intelligentes**
- Hygro-régulantes (Sd variable)
- Simplification mise en œuvre

### Marché

**Tensions approvisionnement**
- Délais PSE, XPS : 4-8 semaines
- Laines minérales : stock tendu périodes fortes
- Anticiper commandes grandes surfaces

**Inflation**
- +6-10% isolants 2023-2025
- PUR/PIR particulièrement touchés
- Biosourcés prix plus stables

**Aides publiques**
- Enveloppes limitées (ruptures possibles)
- Dépôt dossiers avant travaux impératif
- Cumuler aides (MPR + CEE + éco-PTZ)
