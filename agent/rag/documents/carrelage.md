## TEMPS MOYENS DE RÉALISATION

### Préparation
- **Dépose carrelage ancien** : 0,3-0,5 h/m²
- **Ragréage autolissant** : 0,2-0,3 h/m² + séchage 24-48h
- **Primaire d'accrochage** : 0,1 h/m² + séchage 2-4h
- **Création chape** : 0,4-0,6 h/m² + séchage 3-7 jours

### Pose
- **Carreaux 20x20 à 40x40** : 0,4-0,6 h/m²
- **Carreaux 60x60** : 0,5-0,7 h/m²
- **Grand format >75x75** : 0,7-1,0 h/m²
- **Mosaïque** : 1,0-1,5 h/m²
- **Faïence murale** : 0,5-0,8 h/m²
- **Séchage avant jointement** : 24-48h minimum

### Finitions
- **Jointement** : 0,2-0,3 h/m²
- **Nettoyage joints** : 0,15-0,25 h/m²
- **Séchage joints avant utilisation** : 24-72h selon produit

### Projet type (salle de bain 6m²)
- **Durée totale** : 3-5 jours ouvrés
- **Dépose + préparation** : 1 jour
- **Pose sol + murs** : 1-2 jours
- **Jointement + finitions** : 0,5-1 jour
- **Séchages entre étapes** : inclus dans planning

## MATÉRIAUX COURANTS

### Types de carreaux
1. **Grès cérame émaillé**
   - Usage intérieur courant
   - Large choix décoratif
   - Résistance PEI 3-4
   
2. **Grès cérame pleine masse**
   - Intérieur/extérieur
   - Haute résistance (PEI 5)
   - Passage intensif
   
3. **Faïence**
   - Murs uniquement
   - Nombreux décors
   - Sensible aux chocs
   
4. **Mosaïque**
   - Verre, céramique, pierre
   - Zones humides, piscines
   - Surfaces courbes

### Formats standards
- **Petit format** : 10x10, 15x15, 20x20 cm
- **Format moyen** : 30x30, 40x40, 45x45 cm
- **Grand format** : 60x60, 75x75, 80x80 cm
- **Très grand format** : 120x60, 120x120, 160x80 cm
- **Rectangulaire** : 20x60, 30x60, 25x75 cm

### Colles et mortiers
- **C1** : Colle normale (zones sèches)
- **C2** : Colle améliorée (pièces d'eau)
- **C2S1** : Colle déformable (planchers chauffants)
- **C2E** : Temps ouvert étendu (grand format)

### Joints
- **Joints ciment** : Standard, largeur 2-10mm
- **Joints époxy** : Zones humides, chimie
- **Joints souples** : Angles, désolidarisation
- **Couleurs** : Blanc, gris, beige, noir + teintes

## ERREURS FRÉQUENTES

### Préparation
❌ **Support non préparé**
- Surface poussiéreuse, friable
- Défauts de planéité >5mm
- → Décollement, fissures

❌ **Absence de primaire**
- Sur supports poreux/fermés
- → Mauvaise adhérence

❌ **Ragréage insuffisant**
- Planéité non respectée
- → Carreaux instables, bruits

### Pose
❌ **Mauvais choix de colle**
- C1 sur plancher chauffant
- Colle non flexible sur support déformable
- → Décollements

❌ **Encollage incomplet**
- Simple encollage sur grand format
- → Sonorités creuses, casse

❌ **Non-respect du temps ouvert**
- Colle en surface sèche
- → Perte d'adhérence

❌ **Joints de dilatation absents**
- Grandes surfaces >40m²
- Passages intérieur/extérieur
- → Fissures, soulèvements

### Jointement
❌ **Jointement prématuré**
- Colle pas sèche
- → Affaissements, taches

❌ **Largeur joint insuffisante**
- <3mm pour carreaux rectifiés
- → Tensions, éclatements

❌ **Nettoyage tardif**
- Joints durcis
- → Voile ciment permanent

## CONSEILS PRATIQUES

### Avant les travaux
✓ Vérifier planéité support (règle 2m, tolérance 5mm)
✓ Contrôler humidité résiduelle (<3% chape)
✓ Commander 10% de surplus (chutes + casse)
✓ Prévoir carreaux de même lot/calibre
✓ Acclimater carreaux 24h sur chantier

### Pendant la pose
✓ Étaler colle par zones (temps ouvert 15-30min)
✓ Double encollage pour carreaux >30x30
✓ Utiliser croisillons adaptés (2-3-5mm)
✓ Vérifier planéité tous les 3-4 carreaux
✓ Battre les carreaux au maillet
✓ Nettoyer joints avant séchage colle

### Protection et finitions
✓ Protéger sol carrelé frais (cartons)
✓ Ne pas marcher avant 24h minimum
✓ Joints périphériques en silicone souple
✓ Traiter joints si nécessaire (hydrofuge)
✓ Nettoyer voile ciment avec produit adapté

### Zones spécifiques
**Douche italienne**
- Pente 2% vers évacuation
- Étanchéité sous carrelage (SEL)
- Carreaux antidérapants (R10 minimum)

**Plancher chauffant**
- Colle C2S1 minimum
- Mise en chauffe progressive après 3 semaines
- Joints souples périphériques

**Extérieur/terrasse**
- Grès cérame pleine masse gel
- Pente écoulement 1,5-2%
- Joints larges (5-8mm)

## MINI-RÉSUMÉS DTU

### DTU 52.2 (Carrelage scellé)
**Rarement utilisé aujourd'hui - carrelage traditionnel scellé**
- Épaisseur minimale mortier de pose : 15mm
- Support : dalle béton, chape
- Délai avant pose sur dalle neuve : 2 mois

### DTU 52.1 (Carrelage collé - intérieur)
**Référence principale pour pose collée**

**Supports admis**
- Dalle béton, chape ciment
- Planchers chauffants
- Ancien carrelage (si sain)
- Plaques de plâtre hydrofuges (murs)

**Épaisseur de colle**
- Minimum 3mm sous carreau
- Double encollage si carreau >400cm² (20x20)

**Joints de fractionnement**
- Maximum 40m² sans joint
- Passage intérieur/extérieur
- Changement de nature de support

**Largeur joints**
- Minimum 2mm (carreaux rectifiés)
- Recommandé 3-5mm (carreaux standards)
- 5-10mm extérieur

### DTU 52.10 (Carrelage extérieur)
**Terrasses, balcons, zones extérieures**

**Pente obligatoire**
- Minimum 1,5% (terrasse accessible)
- 2% recommandé

**Matériaux**
- Grès cérame pleine masse gel
- Absorption eau <3%
- Résistance glissance R10 minimum

**Joints**
- Largeur 5-8mm minimum
- Joints de dilatation max 25m²
- Périphériques obligatoires

**Étanchéité**
- SEL (Système d'Étanchéité Liquide) sous carrelage
- Bande d'étanchéité angles/relevés

### DTU 43.1 (Étanchéité terrasses)
**Complément pour zones humides**
- Étanchéité sous carrelage obligatoire
- Recoupements 10cm minimum
- Relevés 15cm minimum

## RATIOS MAIN-D'ŒUVRE / MATÉRIAUX

### Répartition coûts moyens

**Projet standard (grès cérame 30€/m²)**
- Carreaux : 35-40%
- Colle, joints, primaire : 10-15%
- Main-d'œuvre pose : 45-50%
- Préparation support : 5-10%

**Exemple chiffrage 20m² sol**
- Carreaux (30€/m²) : 600€
- Fournitures pose : 200€
- Main-d'œuvre : 900€
- **Total : 1700€** (85€/m²)

### Ratios temps/surface

**Main-d'œuvre représente**
- 1,5 à 2x le coût matériaux (pose standard)
- 2 à 3x le coût matériaux (pose complexe)

**Productivité moyenne**
- Carreleur professionnel : 10-15m²/jour (pose standard)
- Grand format : 8-12m²/jour
- Mosaïque : 4-6m²/jour

### Consommation matériaux

**Colle**
- 3-5 kg/m² (simple encollage, peigne 6-9mm)
- 5-8 kg/m² (double encollage, grand format)

**Joint**
- Formule : Longueur x Largeur x Épaisseur x Densité x Coeff 1,6
- Exemple 30x30, joint 3mm, ép 8mm : ~1,5 kg/m²
- Grand format 60x60, joint 3mm : ~0,8 kg/m²

**Rendement indicatif**
- 1 sac colle 25kg : 5-8 m²
- 1 sac joint 5kg : 3-5 m²

### Facteurs multiplicateurs

**Prix augmentent avec**
- Format >60x60 : +20-40%
- Pose diagonale : +15-25%
- Mosaïque : +80-100%
- Préparation support difficile : +30-50%
- Accessibilité réduite : +10-20%

**Temps augmentent avec**
- Première pose (non professionnel) : x2-3
- Découpes nombreuses : +30-50%
- Motifs décoratifs : +50-100%
- Étages sans ascenseur : +10% par étage

## POINTS DE VIGILANCE 2025-2026

### Évolutions normatives
- Renforcement exigences RE2020 (planchers chauffants)
- Étanchéité renforcée zones humides
- Traçabilité fiches techniques produits

### Tendances marché
- Grand et très grand format en progression
- Carreaux minces (3-5mm) en développement
- Colles à prise rapide (pose + joint 1 jour)
- Joints époxy colorés

### Prix matières premières
- Inflation 2023-2025 : +8-12% sur carreaux
- Énergie impacte coûts transport
- Anticiper délais approvisionnement (4-8 semaines)
