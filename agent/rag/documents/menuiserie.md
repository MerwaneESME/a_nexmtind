## MATÉRIAUX COURANTS

### Matériaux châssis

**PVC**
- **Avantages** : Prix, entretien nul, isolation
- **Inconvénients** : Esthétique limitée, déformation chaleur
- **Durée vie** : 25-30 ans
- **Usage** : Résidentiel courant

**Aluminium**
- **Avantages** : Résistant, fins profils, moderne
- **Inconvénients** : Prix, pont thermique (si sans RPT)
- **Durée vie** : 30-40 ans
- **Usage** : Grandes baies, architecture contemporaine
- **Impératif** : À Rupture Pont Thermique (RPT)

**Bois**
- **Avantages** : Esthétique, écologique, isolation
- **Inconvénients** : Entretien, prix
- **Durée vie** : 30-50 ans (si entretenu)
- **Essences** : Pin, chêne, exotiques (moabi, meranti)
- **Traitement** : Lasure tous 5-8 ans

**Mixte bois/alu**
- **Avantages** : Esthétique bois intérieur, alu extérieur sans entretien
- **Inconvénients** : Prix élevé
- **Durée vie** : 40-50 ans
- **Usage** : Haut de gamme, maisons architecte

### Vitrages

**Double vitrage standard (4/16/4)**
- **Ug** : 2,8 W/m².K
- **Usage** : Minimal, non éligible aides

**Double vitrage VIR (4/16/4 FE)**
- **Ug** : 1,1-1,3 W/m².K
- **VIR** : Vitrage Isolation Renforcée (couche FE)
- **Usage** : Standard actuel, éligible aides

**Double vitrage haute performance**
- **Ug** : 0,8-1,0 W/m².K
- **Gaz argon + 2 couches FE**
- **Usage** : RT2012, RE2020

**Triple vitrage (4/12/4/12/4)**
- **Ug** : 0,5-0,8 W/m².K
- **Avantages** : Isolation maximale
- **Inconvénients** : Poids, apports solaires réduits
- **Usage** : Maisons passives, nord, montagne

**Options vitrages**
- **Phonique** : Asymétrique (10/10/4), Rw 35-40 dB
- **Sécurité** : Feuilleté retardateur effraction
- **Contrôle solaire** : Facteur solaire réduit

### Performances thermiques

**Coefficient Uw (fenêtre complète)**
- **Réglementation** : Uw ≤ 1,3 W/m².K (aides)
- **RE2020** : Uw ≤ 1,4 W/m².K
- **Bon** : Uw 1,2-1,4 W/m².K
- **Très bon** : Uw 0,8-1,2 W/m².K
- **Passif** : Uw ≤ 0,8 W/m².K

**Facteur solaire (Sw)**
- **Élevé** : Sw > 0,5 (apports gratuits)
- **Moyen** : Sw 0,4-0,5
- **Faible** : Sw < 0,4 (contrôle solaire)

**Perméabilité air (AEV)**
- **A** : Perméabilité air (A*4 excellent)
- **E** : Étanchéité eau (E*9 excellent)
- **V** : Résistance vent (V*A4 excellent)

### Quincaillerie

**Fermetures**
- **Crémone** : 2-3 points (standard)
- **Multi-points** : 5-7 points (sécurité)
- **Oscillo-battant** : Porte-fenêtre

**Poignées**
- **Standard** : 15-40€
- **Design** : 50-150€
- **Sécurité** : À clé, anti-effraction

**Seuils**
- **Alu à rupture** : Standard
- **PMR** : Seuil plat (<20mm)
- **Classement** : TH2, TH3 selon exposition

## ERREURS FRÉQUENTES

### Dimensionnement

❌ **Mauvaise prise de cotes**
- Oubli jeu de pose (10-20mm)
- → Fenêtre trop grande, impossible à poser

❌ **Sens ouverture inadapté**
- Ouvrant vers mur intérieur
- → Gêne fonctionnelle

❌ **Sécurité ignorée**
- Ouvrant > 90cm sans barre d'appui
- → Danger chute

### Pose

❌ **Calage insuffisant**
- Cales absentes ou mal positionnées
- → Déformation châssis, mauvaise étanchéité

❌ **Étanchéité défaillante**
- Mousse expansive seule
- Joint SNJF absent
- → Infiltrations, déperditions

❌ **Fixations inadaptées**
- Vis trop courtes
- Nombre insuffisant (espacées >60cm)
- → Arrachement, déformation

❌ **Absence pare-vapeur/membrane**
- Liaison dormant/mur non traitée
- → Condensation, pont thermique

### Réglages

❌ **Défaut d'aplomb et niveau**
- Châssis vrillé
- → Mauvaise fermeture, usure prématurée

❌ **Jeu ouvrant/dormant irrégulier**
- Compression joint inégale
- → Infiltrations air, eau

❌ **Quincaillerie mal réglée**
- Points de fermeture désalignés
- → Forçage, casse

### Finitions

❌ **Joint silicone oublié**
- Entre appui et dormant
- → Infiltrations

❌ **Bavettes et rejets eau absents**
- Traverses basses, appuis
- → Stagnation, infiltrations

❌ **Tapées non isolées**
- Retours tableaux non traités
- → Ponts thermiques

## CONSEILS PRATIQUES

### Avant achat

✓ **Vérifier performances**
- Uw ≤ 1,3 pour aides
- AEV selon exposition
- Certification NF ou Cekal

✓ **Choisir matériau adapté**
- PVC : Bon rapport qualité/prix
- Alu : Grandes baies, moderne
- Bois : Charme, cachet
- Mixte : Compromis luxe

✓ **Dimensionner correctement**
- Mesurer tableau (hxl) 3 fois
- Déduire jeu pose 10-20mm
- Vérifier sens ouverture

✓ **Anticiper délais**
- Sur-mesure : 4-8 semaines
- Couleurs spéciales : +2 semaines

### Pendant pose

✓ **Préparation support**
- Tableau propre, sain
- Planéité vérifiée
- Rebouchage fissures

✓ **Calage précis**
- Cales 25-30cm des angles
- Cales tous 60cm maxi
- Vérifier aplomb, niveau, équerrage

✓ **Fixations**
- Pattes ou vis tous 60cm maxi
- 15cm minimum des angles
- Profondeur ancrage 60mm mini

✓ **Étanchéité**
- Joint SNJF (Seuil, Nord, Jambage, Façade)
- Mousse faible expansion (coef 1,3)
- Membrane pare-vapeur intérieur
- Membrane HPV extérieur

### Finitions

✓ **Réglages**
- Compression régulière joint
- Fermeture sans forçage
- Jeu ouvrant/dormant 4-6mm

✓ **Joints silicone**
- Appui/dormant impératif
- Lissage soigné
- Éviter période gel

✓ **Habillages**
- Appuis alu avec pente 10%
- Tapées isolées
- Encadrements si souhaité

✓ **Notices**
- Conserver documentation
- Notice entretien

### Entretien

**PVC et alu**
- Nettoyage savon doux 2x/an
- Lubrification ferrures 1x/an
- Drainage vérifier débouché

**Bois**
- Lasure tous 5-8 ans (climat)
- Vérifier joints 1x/an
- Retouche ponctuelle si besoin

**Tous matériaux**
- Graissage gonds, crémones
- Nettoyage rails coulissants
- Vérification étanchéité

## MINI-RÉSUMÉS DTU

### DTU 36.5 (Menuiseries extérieures)

**Supports**
- Maçonnerie, béton, ossature bois
- Planéité 7mm/2m
- Résistance arrachement suffisante

**Jeu de pose**
- **Largeur** : 10-20mm (15mm moyen)
- **Hauteur** : 10-20mm
- Permet dilatation, réglage, isolation

**Fixations**
- **Pattes ou vis** : tous 60cm maxi
- **Distance angles** : 15cm mini
- **Profondeur** : 60mm mini dans support
- **Nombre mini** : 3 par montant (L<1,50m)

**Calage**
- **Position** : 25-30cm des angles
- **Espacement** : 60cm maxi
- **Matériau** : Cales PVC, bois dur, ciment
- **Maintien** : Permanent (pas de cales temporaires)

**Étanchéité (Joint SNJF)**
- **S** : Seuil (sous dormant)
- **N** : Nord (traversée haute)
- **J** : Jambages (montants)
- **F** : Façade (liaison mur/châssis)

**Ordre pose**
1. Dépose ancienne menuiserie
2. Nettoyage tableau
3. Calage et fixation provisoire
4. Vérification aplomb, niveau, équerrage
5. Fixation définitive
6. Mousse polyuréthane faible expansion
7. Membranes pare-vapeur/HPV
8. Joints silicone
9. Habillages et finitions

### DTU 37.1 (Menuiseries métalliques)

**Spécificités alu**
- Obligatoire : Rupture Pont Thermique (RPT)
- Profilés : Épaisseur mini 1,5mm
- Traitement surface : Thermolaquage, anodisation

**Performances**
- Uw ≤ 1,4 W/m².K (RE2020)
- AEV selon exposition (mini A*3 E*7 V*A3)

**Drainage**
- Orifices évacuation tous les 60cm
- Diamètre 5mm mini
- Grilles anti-insectes

### NF DTU 36.1 (Menuiseries bois)

**Essence de bois**
- Classes 2 à 4 (résistance humidité)
- **Exotiques** : Moabi, merbau, tauari
- **Résineux** : Pin, douglas (traité)
- **Feuillus** : Chêne, châtaigner

**Traitement**
- **Lasure** : Film protecteur, 5-8 ans
- **Peinture** : Opaque, 7-10 ans
- **Saturateur** : Aspect naturel, 2-3 ans

**Épaisseur minimale**
- Ouvrant : 45mm
- Dormant : 58-68mm

**Entretien**
- Contrôle annuel
- Retouche locale si besoin
- Réfection complète 5-10 ans

### CPT 3560 (Pose en rénovation)

**Rénovation sur dormant**
- Ancien dormant sain et plan
- Recouvrement mini 20mm
- Conservation des fixations

**Tapée d'isolation**
- Obligatoire si épaisseur > 120mm
- Retour 10cm mini tableaux
- Matériau isolant (PSE, PUR)

**Appuis de fenêtre**
- Pente 10% mini vers extérieur
- Débord 3-4cm
- Rejet d'eau goutte à goutte
